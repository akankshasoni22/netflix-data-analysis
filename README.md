# netflix-data-analysis
Netflix Data Analysis using Python, Pandas, Matplotlib, and Seaborn to explore content trends, genre popularity, top-producing countries, release patterns, and actor/director insights from 8,800+ Netflix titles.

## Project Overview

This project analyzes the Netflix Titles dataset containing over 8,800 movies and TV shows. The objective is to perform Exploratory Data Analysis (EDA), identify content trends, and generate business insights using Python.

## Dataset Information

* Total Records: 8,807
* Features: 12
* Content Types: Movies and TV Shows

## Objectives

* Analyze Movies vs TV Shows distribution
* Identify top content-producing countries
* Determine the best month and week for content releases
* Analyze top actors and directors
* Explore genre popularity
* Study Netflix content addition trends
* Generate business insights from data

## Key Insights

* Movies dominate Netflix's content library.
* The United States is the leading content-producing country.
* Drama, International Movies, and Comedy are among the most common genres.
* Content additions show clear seasonal trends.
* A small number of actors and directors contribute to a significant portion of Netflix titles.
